📊 Excel Sales Dashboard Project
This repository contains an interactive Sales Dashboard built in Microsoft Excel.
The dashboard combines Pivot Tables, Pivot Charts, and a clean layout to visualize key business metrics such as monthly revenue trends and regional sales performance.

✅ Features
1. Pivot Table

Displays total revenue and units sold for each month.
Provides an aggregated summary for quick insights.
Automatically updates when source data is refreshed.

2. Interactive Charts
Revenue by Month

A line chart showing monthly revenue trends.
Helps track growth patterns and seasonal variations.

Sales by Region

A column chart comparing sales totals across regions.
Includes both revenue and units sold metrics.

3. Dashboard Layout

Clean, user‑friendly layout combining:

A pivot table panel
Two graphical panels
Clear section labels ("Dashboard", "Pivot Table", "Graphs")


Designed for professional presentations or KPI reporting.
